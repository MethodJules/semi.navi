Bibliography & Citation Endnote XML

Provides import and export features for Endnote XML format.

Endnote XML example

<XML>
    <RECORDS>
        <RECORD>
            <REFERENCE_TYPE>10</REFERENCE_TYPE>
            <YEAR>1959</YEAR>
            <TITLE>The Works of John Smith</TITLE>
            <AUTHORS>
                <AUTHOR>John Smith </AUTHOR>
            </AUTHORS>
        </RECORD>
    </RECORDS>
</XML>
