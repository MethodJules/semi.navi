About

Bibliography & Citation Entity

Implements storage for bibliography data as Drupal entities: bibliography, contributor and keyword. Bibliography entity can be rendered as citations, exported and imported.

Reference Entity

Reference entities stores all bibliographic data. There are more than 30 predefined types (bundles) e.g. artwork, book, etc. These bundles are configurable: you can add your own bundles via the administration UI. Bundles are fieldable.

Contributor Entity

Contributor entities store info about authors. Authors may be assigned to a role and a category. You can manage a list of roles and categories via the administration UI.

Merge Feature

Important feature is deduplication. If you have duplicated author or keyword records, you can merge them into a single entity. It can be done manually (via administration views action) or automatically during import process.

Settings pages

  • Reference - set display settings of reference.
  • Links - manage links of reference.
  • Types - edit or delete a predefined reference types.
  • Contributor - set full name pattern of contributor.
  • Contributor category - edit or delete a predefined contributor categories.
  • Contributor role - edit or delete a predefined contributor roles.
  • CSL map - set mapping CSL types and fields to reference type and fields.
